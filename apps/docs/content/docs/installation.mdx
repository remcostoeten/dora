# Installation

Dora is distributed as a native desktop application.

## Development

To build Dora from source, you need Rust and Node.js installed.

<Callout type="info">
Ensure you have `pnpm` installed and `cargo` (Rust toolchain).
</Callout>

1. **Clone the repository**:
   ```bash
   git clone https://github.com/remcostoeten/dora.git
   ```

2. **Install dependencies**:
   ```bash
   pnpm install
   ```

3. **Run the desktop app**:
   ```bash
   pnpm desktop:dev
   ```

4. **Run these docs**:
   ```bash
   pnpm docs:dev
   ```
