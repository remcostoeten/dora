# Connect YugabyteDB

YugabyteDB is a distributed SQL database whose YSQL API speaks the PostgreSQL
wire protocol. Dora connects to it as a Postgres host.

## With a connection string

1. In **YugabyteDB Aeon** (Managed) open your cluster and click **Connect →
   Connect to your application**. Add your machine's IP to the cluster's
   **IP allow list** first.
2. Copy the connection parameters and assemble a Postgres URL using the YSQL
   port (`5433`):

   ```text
   postgresql://<user>:<password>@<host>.yugabyte.cloud:5433/yugabyte?sslmode=require
   ```

3. In Dora press `Mod+Shift+N`, toggle **Use connection string**, paste the URL,
   **Test connection**, and **Save**.

<Callout type='info'>YugabyteDB's YSQL port is `5433`, not the usual Postgres `5432`. Managed clusters require SSL. Keep `sslmode=require` in the URL or enable the SSL toggle. Download the cluster CA certificate if your SSL mode needs to verify it.</Callout>
