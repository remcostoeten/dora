---
title: Connect Google Cloud SQL
description: Connect Dora to a Google Cloud SQL database (PostgreSQL or MySQL).
---

Cloud SQL hosts PostgreSQL and MySQL on Google Cloud. Dora detects the engine
from the connection string and you assemble the URL from the instance's public
IP.

## With a public IP

1. In the Google Cloud console open **SQL** and select your instance.
2. On the **Overview** page, under **Connect to this instance**, note the
   **Public IP address** and add your machine's IP under
   **Connections → Networking → Authorized networks**.
3. Build a URL with a database user and the public IP:

   ```text
   postgresql://<user>:<password>@<public-ip>:5432/<db>
   ```

   For MySQL use `mysql://` and port `3306`.

4. In Dora press `Mod+Shift+N`, toggle **Use connection string**, paste the URL,
   **Test connection**, and **Save**. Enable the **SSL** toggle if the instance
   requires TLS.

<Callout type='info'>Prefer not to expose a public IP? Run the **Cloud SQL Auth Proxy** locally and point Dora at `127.0.0.1` on the proxy's port. Dora connects to it like a local database.</Callout>
