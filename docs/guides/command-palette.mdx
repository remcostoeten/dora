## Command palette

Press `Mod+K` (⌘ on macOS, Ctrl elsewhere) anywhere in Dora to open the command
palette. From it you can navigate to any view, switch connections, jump to a
table, reopen a query from history, open a snippet, manage Docker containers, or
go to settings. Start typing to filter; a breadcrumb shows where you are.

## Keyboard shortcuts

`Mod` is ⌘ on macOS and Ctrl on Windows / Linux.

### Query

| Action | Shortcut |
| --- | --- |
| Run query | `Mod+Enter` |
| Run selected SQL | `Mod+Shift+Enter` |
| Format SQL | `Mod+Shift+F` |
| Open query history | `Mod+H` |
| Save script | `Mod+S` |
| Find in editor | `Mod+F` |
| Toggle line comment | `Mod+/` |

### Navigation

| Action | Shortcut |
| --- | --- |
| Command palette | `Mod+K` |
| Add connection | `Mod+Shift+N` |
| Toggle sidebar | `Mod+B` |
| Settings | `Mod+,` |
| Close current tab | `Mod+W` |
| Close dialog / cancel | `Escape` |

### Data Studio

| Action | Shortcut |
| --- | --- |
| Refresh table | `Mod+R` |
| Filter rows | `Mod+F` |
| Insert row | `Mod+Shift+I` |
| Delete selected row(s) | `Backspace` / `Delete` |
| Export table | `Mod+E` |
| Live monitor | `Mod+Shift+M` |

### Go to (press `G`, then the next key)

| Action | Shortcut |
| --- | --- |
| Dashboard | `G` then `D` |
| Settings | `G` then `S` |
| Connections | `G` then `C` |

## Customize shortcuts

Open **Settings → Keyboard Shortcuts** to rebind anything. Click a shortcut's
binding, press the new combination, and Dora records it. You can search by
action or key, filter by category, and reset to defaults. Dora warns you about
conflicts. Your overrides are saved across sessions.
