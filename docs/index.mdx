# Dora docs

Dora is a native desktop SQL client. It connects to local files, hosted
databases, and team infrastructure, then gives you a fast table browser, a SQL
console with autocomplete, type-safe ORM runners, a schema diagram, and an AI
assistant, in one window.

New here? Start with the [getting started walkthrough](/docs/getting-started),
then [install Dora](/docs/installation) for your platform.

## Connect a database

Dora connects to any PostgreSQL, MySQL, MariaDB, CockroachDB, SQLite, DuckDB, or
libSQL host. Paste a connection string and Dora detects the engine and SSL mode
automatically. See [connection basics](/docs/connect).
